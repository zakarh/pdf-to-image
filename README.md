# pdf-to-image
Convert a PDF document into a ZIP of JPGs

## Technologies
- Python 3
- Poppler
- Docker

## Local Setup

1. Install Docker.
2. Build image from Dockerfile.
3. Run container from image.

## Using the Application

1. Visit http://localhost:5000.
2. Browse and select a PDF.
3. Press Submit and the PDF will be uploaded for conversion.
4. Once done wait for the ZIP file to download.
4. Unzip the archive to access the JPG images.
